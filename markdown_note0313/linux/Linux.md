### Wget && curl ###

- wget是个专职的下载利器，简单，专一，极致
- curl
	- 利用URL语法在命令行方式下工作的开源文件传输工具
	- 可以下载，但是长项不在于下载，而在于模拟提交web数据，POST/GET请求，调试网页，等等。
